# 虚拟DOM

虚拟DOM介绍

全面理解虚拟DOM，实现虚拟DOM

一种可以预先通过js进行各种计算, 把最终的DOM操作计算出来的并优化的技术, 由于这个DOM操作属于预处理操作, 并没有真实的操作DOM, 所以叫做虚拟DOM。

Vue.js 2.0引入 Virtual DOM,比Vue.js 1.0的初始渲染速度提升了2-4倍，并大大降低了内存消耗。
