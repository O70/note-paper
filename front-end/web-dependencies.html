# 依赖管理

# 部分依赖

  • axios: http请求库, 可以用在浏览器和 node.js 中

  • vue: 核心库, 只关注视图层

  • vue-loader: 一个webpack的loader，它允许你以一种名为单文件组件(SFCs)的格式撰写Vue组件

  • vue-router: 路由管理器

  • vuex: 一个专为Vue.js应用程序开发的 状态管理模式

  • element-ui: UI框架(2019最受欢迎的前端7个UI框架)

  • babel*: js编译器, 主要用于将ES6+的代码转为向后兼容的js语法(语法转换)

  • css-loader: CSS 相关

    • 浅谈css预处理器，Sass、Less和Stylus
    • 浅谈 CSS 预处理器（一）：为什么要使用预处理器？

  • eslint: 属于一种QA工具，是一个ECMAScript/JavaScript语法规则和代码风格的检查工具，可以用来保证写出语法正确、风格统一的代码

  • webpack*: 用于开发及打包
